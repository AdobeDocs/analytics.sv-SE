# Webbläsartyper

Avser den typ av webbläsare som används av besökaren, t.ex. Internet Explorer, Mozilla Firefox, Google Chrome och så vidare.

Webbläsartyper som ännu inte identifierats visas som&quot;ospecificerade&quot;.

Exempel:

| Google |

| Mozilla |
| Microsoft |
| Apple |
| Opera |
| Ospecificerad |
| BlackBerry |
| UC Web |
| Övriga |
| Nokia |
