# Informationstjänster

Grupperar mobila enheter baserat på nyhetstjänsterna (Channel I/B, EZ News Flash, W+INFO osv.). som de stöder. När en besökare använder en mobil enhet med stöd för nyhetstjänster för att få tillgång till din webbplats ökar rapporten om informationstjänster med minst en. Om telefonen har stöd för flera nyhetstjänster ökar rapportens storlek för var och en av grupperna. Därför kan summan av grupperna vara större än summan som visas längst ned i rapporten.
