# Report Builder - översikt

Report Builder är ett tillägg för Microsoft Excel. Här kan du skapa anpassade förfrågningar från rapporter och analysdata, som du kan infoga i dina Excel-kalkylblad. Begäranden kan referera till celler dynamiskt i kalkylbladet, och du kan uppdatera och anpassa hur Report Builder visar data.
